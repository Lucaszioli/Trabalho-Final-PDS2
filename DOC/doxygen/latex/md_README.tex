O projeto desenvolvido, com inspiração no clássico Pokemón Stadium, se baseia em um jogo de luta de arena de player vs bot.\hypertarget{md_README_autotoc_md2}{}\doxysection{\texorpdfstring{Como jogar}{Como jogar}}\label{md_README_autotoc_md2}

\begin{DoxyEnumerate}
\item {\bfseries{Início do \doxylink{classJogo}{Jogo}}}\+:
\begin{DoxyItemize}
\item Abrindo o jogo insira o seu nome.
\end{DoxyItemize}
\item {\bfseries{Escolha dos Monstrinhos}}\+:
\begin{DoxyItemize}
\item Você começa com 12 moedas.
\item Use as moedas para escolher 4 monstrinhos, cada um com um custo entre 1 e 5 moedas.
\item Os monstrinhos são os seus aliados na batalha!
\end{DoxyItemize}
\item {\bfseries{Enfrentando o \doxylink{classTreinador}{Treinador}}}\+:
\begin{DoxyItemize}
\item Após escolher seus monstrinhos, você enfrentará um treinador com seu próprio time de 4 monstrinhos.
\item A batalha começa e você pode escolher entre as seguintes opções\+:
\begin{DoxyItemize}
\item {\bfseries{Atacar}}\+: Escolha um dos 4 ataques do seu monstrinho para atacar o inimigo.
\end{DoxyItemize}
\end{DoxyItemize}
\end{DoxyEnumerate}

{\bfseries{Mudar de \doxylink{classMonstrinho}{Monstrinho}}}\+: Troque o monstrinho atual para um de seus outros monstrinhos para atacar.
\begin{DoxyItemize}
\item {\bfseries{Usar um \doxylink{classItem}{Item}}}\+: Use um item especial em seu monstrinho atual para melhorar suas chances na batalha.
\end{DoxyItemize}

{\bfseries{Ataques dos Monstrinhos}}\+:
\begin{DoxyItemize}
\item Cada monstrinho possui 4 ataques diferentes.
\item Escolha o ataque que você deseja usar para causar dano ao inimigo.
\end{DoxyItemize}

{\bfseries{Itens do jogo}}\+:
\begin{DoxyItemize}
\item {\bfseries{\doxylink{classEstamina}{Estamina}}}\+: Recupera a estamina de um ataque!
\item {\bfseries{\doxylink{classRevive}{Revive}}}\+: \doxylink{classRevive}{Revive} um monstrinho já derrotado!
\item {\bfseries{\doxylink{classCura}{Cura}}}\+: \doxylink{classCura}{Cura} o HP de um monstrinho!
\end{DoxyItemize}

{\bfseries{Fim do \doxylink{classJogo}{Jogo}}}\+:
\begin{DoxyItemize}
\item O jogo termina quando todos os monstrinhos de um dos times são derrotados.
\end{DoxyItemize}\hypertarget{md_README_autotoc_md3}{}\doxysection{\texorpdfstring{Dicas}{Dicas}}\label{md_README_autotoc_md3}

\begin{DoxyItemize}
\item {\bfseries{Moedas}}\+: Gerencie suas moedas com sabedoria para garantir que você tenha os melhores monstrinhos para a batalha.
\item {\bfseries{Estratégia}}\+: Use a estratégia para escolher quando mudar de monstrinho ou usar itens para obter vantagem na batalha.
\item {\bfseries{Tipos}}\+: Lembre-\/se que dependendo do tipo do seu ataque e do tipo do monstrinho inimigo você pode ter vantagem!
